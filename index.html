HTML

HTML (Hypertext Markup Language) is one of the three main languages used in web development. HTML is used to create the structure of a webpage. There are two elements that are specially important to understand how the structure of a webpage works:

  • Head: it contains information or metadata. Helps the browser to render the page correctly, the elements cant be seen by the user on the webpage.
  • Body: it contains the content that is shown to the user. All the elements that we want to be seen and interacted with by the user must be placed inside this element.

DOCTYPE html declaration

  • This is the HTML document type declaration, this is the first line of code required in every HTML or XHTML document.
  • This ensures that the web page is parsed the same way by different web browsers.

Tags

  • Tags are used to indicate the beggining and the end of an element in your HTML document.
  • We use angle brackets to open and close tags.
