HTML


HTML (Hypertext Markup Language) is one of the three main languages used in web development. HTML is used to create the structure of a webpage. There are two elements that are specially important to understand how the structure of a webpage works:


  • Head: it contains information or metadata. Helps the browser to render the page correctly, the elements cant be seen by the user on the webpage.
  • Body: it contains the content that is shown to the user. All the elements that we want to be seen and interacted with by the user must be placed inside this element.

DOCTYPE html declaration


  • This is the HTML document type declaration, this is the first line of code required in every HTML or XHTML document.
  • This ensures that the web page is parsed the same way by different web browsers.

Tags


  • Tags are used to indicate the beggining and the end of an element in your HTML document.
  • We use angle brackets to open and close tags.

CSS Cascading Style Sheets


There are three different ways to style a webpage using Cascading Style Sheets (CSS): inline CSS, internal CSS style sheet, and external CSS style sheet.


  • Inline CSS: It is used to style directly to an element in the HTML file. They are applyed to a specific HTML tag.
  • Internal CSS: It contains CSS rules in the head section of the HTML file and they only apply to that page.
  • External CSS: This is the most common way to style a webpage because it allowa other developers to make design changes easier.

CSS Rules:


  • A selector defines the element or attributes where the rules or declarations will be applied.
  • Declarations contain two important components which are the CSS property we want to apply and the value of the property.

Everything on CSS has a box around it, there are to categories of boxes: block boxes and inline boxes. Every box is divided into different parts:


  • Content box: it is the area where your content is going to be displayed.
  • Padding box: it sits around the content as white space.
  • Border box: it wraps the content and any padding.
  • Margin box: this is the outermost layer, wrapping the content, padding, and border as whitespace between this box and other elements.

Git
